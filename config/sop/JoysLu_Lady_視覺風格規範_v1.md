# JoysLu Lady 賣場視覺風格規範

**版本:** v1.0
**建立日期:** 2026-05-18
**用途:** 統一賣場視覺呈現、避免風格混亂
**重要說明:** 本規範僅為「視覺一致性管理」、非品牌建立投資

---

## 一、規範目的

### 1.1 為什麼需要這份規範

賣場現況的問題:
- 各區塊風格混亂、缺乏一致性
- 商品圖、活動圖、首頁視覺各自獨立、像雜貨店拼貼
- 客人視覺感受雜亂、缺少信任感

這份規範解決的問題:
- 統一賣場主色調
- 統一字體規範
- 統一圖示風格
- 讓客人感受「賣場有經營、有條理」

### 1.2 規範範圍

✅ **本規範涵蓋:**
- 賣場色彩系統
- 字體選擇規範
- 圖示風格規範
- 視覺應用範例

❌ **本規範不涵蓋:**
- 商品包裝升級(不做)
- 售價調整策略(獨立決策)
- 商品拍攝風格升級(不做)
- 完整品牌識別系統(不做)

## 三、配色系統

### 3.1 品牌色票

| 顏色 | 色碼 | 用途 |
|---|---|---|
| 米色(底色) | `#F2ECE6` | 大面積底色、賣場主視覺背景 |
| 淺粉(輔助) | `#E8D9D1` | 輔助色塊、強調區域 |
| 玫瑰粉(主色) | `#D4B5AC` | 標題、重點文字、按鈕 |
| 灰米(中性) | `#B7ACA6` | 次要文字、邊框 |
| 中灰(文字) | `#8E8580` | 主要內文 |
| 深灰(對比) | `#4A4646` | 強對比文字、深色標題 |

### 3.2 配色運用比例

| 顏色用途 | 佔比 | 說明 |
|---|---|---|
| 底色(米色 + 淺粉) | 60-70% | 主視覺主導 |
| 中性(灰米 + 中灰) | 20-25% | 文字、輔助 |
| 強調(玫瑰粉) | 5-10% | 重點、按鈕、標題 |
| 對比(深灰) | 5% | 必要時的強對比 |

### 3.3 顏色搭配原則

- 同一個視覺元素內、最多使用 3 個色塊
- 避免高對比配色(紅綠、黑紅)
- 保持「**溫柔、米粉灰**」的整體氛圍
- 不使用螢光色、純黑、純紅

## 四、字體規範

### 4.1 字體選擇

**中文字體:** 思源柔黑 / Noto Sans TC

**英文字體:** 優雅手寫體(用於 Slogan)+ 標準襯線體(用於正文)

### 4.2 字體階層

| 用途 | 字體 | 大小參考 |
|---|---|---|
| 主標題(賣場名) | 手寫體 / 中文細黑 | 36-48pt |
| Slogan | 手寫體 / 細體中文 | 20-24pt |
| 次標題 | 思源柔黑 Medium | 18-22pt |
| 內文 | 思源柔黑 Regular | 14-16pt |
| 註解 | 思源柔黑 Light | 12-14pt |

### 4.3 字體使用原則

- 標題與內文「至少」要有明顯字級差異
- 同一張圖內最多使用 2 種字體
- 不使用過度花俏的字體(如手寫圓體、毛筆體)
- 中文字體不使用過粗字重(避免笨重感)

## 五、圖示風格規範

### 5.1 圖示風格定義

**統一風格:** 細線條 ‧ 圓潤邊角 ‧ 單色

- 線條粗細:1.5-2pt
- 邊角處理:圓潤、不銳利
- 顏色:單色(玫瑰粉 #D4B5AC 或中灰 #8E8580)
- 不使用:陰影、漸層、立體效果

### 5.2 賣場分類圖示

| 分類 | 圖示元素 |
|---|---|
| 內衣 / BraTop | 內衣輪廓 |
| 內褲 / 安全褲 | 內褲輪廓 |
| 絲襪 / 襪套 | 腿部 + 絲襪輪廓 |
| 襪子 | 短襪 / 中筒襪輪廓 |
| 上衣 | T 恤輪廓 |
| 褲子 | 長褲輪廓 |

### 5.3 圖示尺寸規範

- 圖示框架統一:正方形 1:1
- 圖示佔框架 60-70%(保留留白)
- 多個圖示並列時、視覺重量需平衡

## 六、視覺應用範例

### 6.1 賣場頭圖(Banner)

**結構:**
```
═══════════════════════════════
[Logo: JoysLu Lady]
(手寫體字樣)

闆娘精選 ‧ 內著生活
Enjoy Your Beauty
═══════════════════════════════
```

**設計原則:**
- 底色:米色 #F2ECE6
- 主視覺文字:玫瑰粉 #D4B5AC
- 輔助文字:中灰 #8E8580
- 大量留白、不擁擠
- **不加品牌承諾文字**(如「我們提供的不只是商品」這類)

### 6.2 輪播圖(3-4 張)

**建議內容:**

| 圖 | 主題 | 重點 |
|---|---|---|
| 1 | 會員引導 | 加入會員 + 追蹤賣場、享 9.5 折券 + 30 點 |
| 2 | 主力品類展示 | 內衣 / 內褲 / 絲襪 / BraTop 四大品類 |
| 3 | 全店活動懶人包 | 列出四個主活動規則 |
| 4 | 當季主推 | 季節性主推商品(可隨季節更換) |

**統一風格:**
- 同一配色系統
- 同一字體規範
- 同一圖示風格

### 6.3 自訂商品分類

使用 6-8 個快速入口、每個入口配統一風格圖示:

| 入口 | 對應分類 |
|---|---|
| 內衣 BraTop | 內衣 + BraTop + NUBRA |
| 內褲 安全褲 | 內褲 + 丁字褲 + 安全褲 |
| 絲襪 襪套 | 絲襪 + 連褲襪 |
| 襪子 | 短襪 + 中筒襪 |
| 上衣 | T 恤類(維持模式) |
| 褲子 | 居家褲 + 運動褲(未來擴張) |

### 6.4 詳情頁活動圖

**現有版本維持不動**——

詳情頁已有的「會員 + 追蹤 + 取貨完成換券」整合圖、色系本來就接近本規範(粉紅 + 薄荷綠)、繼續沿用。

### 7.1 立即套用(本月)

| 順位 | 動作 | 預估時間 |
|---|---|---|
| 1 | 賣場頭圖(Banner) | 1-2 小時 |

|---|---|---|

### 7.3 不套用範圍

- ❌ 商品本身的視覺
- ❌ 商品包裝
- ❌ 商品定價策略
- ❌ 客服回覆風格(維持原本即可)

## 八、不該做的事(避免過度品牌化)

### 8.1 不加品牌承諾文字

避免在賣場視覺中出現:
- ❌ 「我們提供的不只是商品」
- ❌ 「品質把關 ‧ 安心選購」
- ❌ 「精選質感 ‧ 貼心服務」
- ❌ 任何「我們承諾...」的文字

**理由:** 賣場目前定位是「現金流機器」、不是「品牌建立」、過度承諾會造成客人期待落差。

### 8.2 不過度堆疊品牌訊號

避免:
- ❌ 完整的品牌理念介紹頁
- ❌ 創辦人故事
- ❌ 品牌使命宣言
- ❌ 「我們的價值」六大主張

**理由:** 統一視覺 ≠ 建立品牌、兩者投資邏輯不同。

### 8.3 不反覆修改規範

- 規範設定後、6 個月內不大改
- 微調僅限「色碼調整」「圖示替換」
- 重大調整需要明確商業理由

## 九、未來規劃

### 9.1 短期(6-12 個月)

- 套用此視覺規範到 Lady 賣場
- 累積一致的視覺資產
- 不擴張到包裝、商品定位

### 9.2 中期(12-24 個月)

如果未來決定「**從 Lady 挑爆款獨立做品牌**」:
- 此視覺規範可保留、給「新獨立品牌」使用
- 新品牌可在此基礎上擴張為完整品牌識別系統
- 與 Lady 賣場視覺脫鉤、各自獨立

### 9.3 長期(24 個月+)

- 視商業發展決定 Lady 視覺是否升級
- 不預設方向、保持彈性
