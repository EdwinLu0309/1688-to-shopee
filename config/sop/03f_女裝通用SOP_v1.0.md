## 一、設計原則（給 Claude Code 看的）

### 1.1 本 SOP 的定位

- ⭐ **預購商品專用** ‧ 不做專屬細分 SOP ‧ 一份通用版打天下
- ⭐ **目標完整度 80%** ‧ 抓到賣場一致性即可 ‧ 不追求 100%
- ⭐ **規則盡量寬鬆** ‧ 廠商有什麼寫什麼 ‧ 沒有就 fallback ‧ 不糾結細節
- ⭐ **未來累積實戰後**：若某子品類量大（如 T 恤 50 款+），再拆專屬 SOP

### 1.2 跑稿核心心法

```
廠商有寫 → 直接用
廠商沒寫 → 套 fallback 或省略
廠商寫得很詳細 → 抓重點寫，不全部塞進去
有疑慮 → flag 給 Edwin，不默默改
```

### 1.3 不要做的事

- ❌ 不要為了「寫滿」而腦補機能（廠商沒寫透氣就不寫透氣）
- ❌ 不要花太多時間糾結用字精準度
- ❌ 不要為了 80% 不到的差異跑多版本給 Edwin 挑
- ❌ 不要在「款式分類」上糾結太久（不確定就走通用方向）

## 四、賣點選擇方向（5 大類通用）

### 4.1 萬用賣點關鍵字（任何子品類都可用）

```
版型方向：修身顯瘦 / 寬鬆遮肉 / Oversize / 合身剪裁 / 顯腿長 / 顯腰瘦
材質方向：親膚柔軟 / 透氣不悶 / 不易皺 / 彈性舒適 / 棉質親膚 / 雪紡飄逸
穿搭方向：百搭基本款 / 一件搞定 / 內搭外穿皆可 / 通勤約會皆宜 / 春夏新款 / 秋冬必備
細節方向：精緻車工 / 立體剪裁 / 質感配色 / 經典版型 / 簡約設計
```

### 4.2 子品類賣點重點方向

| 子品類 | 賣點重點 |
|---|---|
| 上衣 / T 恤 | 版型修身、材質親膚、百搭、內搭外穿、顯瘦 |
| 外套 | 保暖、版型挺、好搭、輕量、不易皺 |
| 褲子 / 短褲 | 顯腿長、修身、彈性、不勒腰、版型剪裁 |
| 裙裝 | 顯瘦、修飾大腿、百搭、版型挺、好穿脫 |
| 連身類 | 一件搞定、顯瘦、版型修飾、百搭 |

### 4.3 ⭐ 賣點寫作核心規則

```
廠商 1688 解析有寫 → 照寫
廠商沒寫 → 套萬用賣點關鍵字（4.1 節）
廠商寫的很細（如「45% 棉 55% 滌綸」）→ 簡化為「棉質混紡」
廠商寫的很模糊（如「韓系」）→ 改為具體「日系簡約 / 韓系修身」
```

⚠️ 預購商品不要寫太具體的功能承諾（如「精準減肥效果」「絕對顯瘦 5kg」）

## 八、尺碼系統判定（女裝通用版 ‧ 寬鬆處理）

### 8.1 三種常見尺碼系統

```
廠商 SKU + 廠商給的資料
     ↓
看 SKU 樣式：
     ↓
├─ SKU 標示 S/M/L/XL + 廠商給體重 / 三圍對照
│   └─ 【體重對照型】← 直接照廠商寫
│
├─ SKU 標示 S/M/L/XL（廠商沒給對照）
│   └─ 【純字母型】← 套用 fallback 通用對照表
│
├─ SKU 標示均碼 / One Size / F
│   └─ 【均碼型】← 廠商給體重就照寫，沒給寫「均碼」即可
│
└─ SKU 標示數字（25/26/27 腰圍 等）
    └─ 【數字尺碼型】← 直接照廠商寫
```

### 8.2 ⭐ Fallback 通用女裝體重對照表（廠商沒給時使用）

```
S 號｜建議體重 40-47 kg
M 號｜建議體重 47-55 kg
L 號｜建議體重 55-62 kg
XL 號｜建議體重 62-70 kg
XXL 號｜建議體重 70-78 kg
```

⚠️ **使用規則**：
- ✅ 廠商**完全沒給**對照 → 套此 fallback
- ❌ 廠商有給 → 永遠以廠商為準（不要用 fallback 覆蓋）
- ⚠️ 此表為**寬版預估** ‧ 不準確時請以實際尺寸為主

### 8.3 寫法範例

**體重對照型（廠商給對照）**：
```
▪ 尺碼：S（40-45 kg）‧ M（45-52 kg）‧ L（52-60 kg）
```

**純字母型（廠商沒給 → 套 fallback）**：
```
▪ 尺碼：S ‧ M ‧ L ‧ XL（建議體重對照 ‧ 詳見商品圖尺寸表）
```

**均碼型**：
```
▪ 尺碼：均碼（建議體重 40-60 kg ‧ 廠商有給才寫範圍）
```

**數字尺碼型（廠商給腰圍）**：
```
▪ 尺碼：25 / 26 / 27 / 28 / 29 / 30（廠商腰圍標示）
```

## 九、溫馨提醒（女裝品類通用 4 點）

### 9.1 標準模板

```
💡【溫馨提醒】

▪ 手工測量誤差 1-3 cm 屬正常
▪ 建議冷水手洗 ‧ 內衣洗衣袋機洗亦可
▪ 深色款建議單獨清洗 ‧ 避免染色
▪ 勿浸泡 ‧ 勿熱水 ‧ 勿烘乾
```

### 9.2 廠商備註優先級

如果廠商有特殊備註（如「版型偏小建議大一號」「易掉色請分開洗」），**第一點換成廠商備註**，往下推：

```
💡【溫馨提醒】

▪ 版型偏小建議大一號購買                  ← 廠商備註優先
▪ 手工測量誤差 1-3 cm 屬正常
▪ 建議冷水手洗 ‧ 內衣洗衣袋機洗亦可
▪ 勿浸泡 ‧ 勿熱水 ‧ 勿烘乾
```

### 9.3 不寫清單

```
❌ 螢幕色差（過度防備感）
❌ 新品有味道（過度防備感）
❌ 廠商「3-6 個月更換」（女裝不適用）
❌ 「適用對象」相關備註（依規範禁寫）
```
